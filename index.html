Leetcode SQL Visualizer

{
  "headers": [
    "Column A",
    "Column B",
    "Column C"
  ],
  "values": [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, null]
  ]
}
"headers" is an array of column names "values" is a nested array of row values Column A contains values 1, 4, 7 Column B contains values 2, 5, 8 Column C contains values 7, 8, null All rows must contain the same number of values
